CAPABILITIES
What PII-Safe does
Six core engines protecting every piece of data through the entire pipeline
01
🔍
PII Detection
Schema-aware detection of 7 entity types — email, IP, SSN, credit card, phone, API key, username — using regex with confidence scoring and risk assessment.
02
📋
Policy Engine
YAML policy-as-code evaluates each request and returns allow, redact, pseudonymize, or block. Rules are hot-reloadable without server restart.
03
⚖️
Compliance Mapper
Maps detected PII to exact legal articles — GDPR Art.17, HIPAA 45 CFR 164.514(b), CCPA §1798.105 — with critical/high/medium severity scoring.
04
🛡️
Injection Shield
Detects 14 prompt injection patterns hidden inside data — instruction overrides, persona hijacks, policy bypasses, role spoofing, encoded payloads.
05
📊
Re-ID Risk Scorer
Calculates residual re-identification risk from field combinations even after partial redaction, based on Latanya Sweeney's anonymization research.
06
📝
Audit Trail
Every transformation is logged with original value, replacement token, action type, and scope ID — enabling authorized re-identification for compliance.
